## Project 2: Titanic Dataset EDA

### Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The objective is to explore passenger demographics, survival patterns, fare distribution, and relationships between different features using statistical analysis and data visualization techniques.

### Features

* Dataset loading and inspection
* Statistical summary of passenger data
* Age distribution analysis
* Gender-wise passenger analysis
* Fare distribution exploration
* Survival rate analysis
* Passenger class comparison
* Age vs Fare relationship analysis
* Data visualization using charts and plots

### Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib

### Dataset Information

The Titanic dataset contains passenger information such as age, gender, passenger class, fare, and survival status. The dataset is widely used for data analysis and machine learning practice.

### Project Workflow

1. Import required libraries
2. Load the Titanic dataset
3. Explore dataset structure and statistics
4. Analyze passenger demographics
5. Visualize age and fare distributions
6. Examine survival patterns
7. Compare passenger classes and gender-based insights
8. Generate meaningful visualizations and conclusions

### Learning Outcomes

* Exploratory Data Analysis (EDA)
* Data Visualization Techniques
* Statistical Analysis using Pandas
* Working with Real-World Datasets
* Data Interpretation and Insight Generation
